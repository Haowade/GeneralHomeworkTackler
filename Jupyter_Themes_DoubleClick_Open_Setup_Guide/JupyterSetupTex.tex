  \subsection[]{Jupyter Installation:}
  
  Make sure you have \textbf{Python} installed And do:\\

  \indent\framebox[1.1\width]{\textit{pip install jupyter}}\\

  After the installation is complete, use the command:\\

  \indent\framebox[1.1\width]{\textit{juyter notebook}}\\

  to start, your browser window should pop up very soon

  \subsection[]{Changing Jupyter Theme:}

  You can install theme using \textbf{Jupyter-Theme} by:\\

  \indent\framebox[1.1\width]{\textit{pip install jupyterthemes}}\\

  You can check the list of themes available with:\\

  \indent\framebox[1.1\width]{\textit{ jt -l\ }}\\

  To install a theme:\\

  \indent\framebox[1.1\width]{\textit{jt -t chesterish}}\\

  \subsection[]{Double-click Startup:}

  You can install \textbf{nbopen} for quick double-click startup:\\

  To Install:\\

  \framebox[1.1\width]{\textit{python3 -m pip install nbopen}}\\

  To integrate to your file manager:\\

  Linux or BSD: \framebox[1.1\width]{\textit{python -m nbopen.install\_xdg}}\\

  Windows: \framebox[1.1\width]{\textit{python -m nbopen.install\_win}}\\

  Mac: \framebox[1.1\width]{\textit{Clone the repository and run ./osx-install.sh}}\\
